## 📖 Description

사람들이 자주 사용하는 LLM에 신뢰성을 더해주는 서비스입니다.

LLM을 잘 활용할 수 있도록 프롬프트를 제공하며, 여러 LLM 간 교차검증 및 출처를 제공합니다.

환각도와 유사도를 검사해 답변이 신뢰성있는지 검사해줍니다.



## ✨ Demo & Main Feature
### 1. 메인 화면
<img width="724" height="442" alt="image" src="https://github.com/user-attachments/assets/d8c3216b-5cdc-437d-bee7-b30f06739c93" />


- 프롬프트 최적화 / AI 교차검증 으로 나눠서 메인 기능 2가지를 강조합니다.



### 2. 프롬프트 최적화 기능
<img width="491" height="258" alt="프롬프트최적화" src="https://github.com/user-attachments/assets/c8911390-3fa4-4a28-9b7c-1c53da7d606f" />


- 자신이 묻고 싶은 질문을 입력하고,

- 사용자 페르소나 및 질문 분야에 대한 도메인을 선택하여 최적의 답변을 얻을 수 있는 프롬프트를 생성해줍니다.


### 3. AI 교차 검증 화면
<img width="491" height="255" alt="교차검증" src="https://github.com/user-attachments/assets/2186b7b2-076a-42d0-8739-be0123d2a46c" />


- 사용자는 직접 입력한 질문이나 최적화된 프롬프트를 이용해 최대 4개의 AI에게 동시에 답변을 얻을 수 있습니다.


### 4. 환각 검증 페이지
<img width="438" height="601" alt="환각검증" src="https://github.com/user-attachments/assets/d1f3f9aa-75c4-4675-b3e4-106b67c14fde" />


- AI 교차검증 페이지에서 얻은 답변을 기반으로 각 AI 답변끼리의 유사도, 환각 의심도 수치를 한눈에 파악할 수 있는 페이지입니다.

- 툴팁을 통해 환각의심도에 대한 정의를 제공합니다.
  
- 각 AI 가 해당 답변에 대한 정보를 어디서 참고했는지 외부 자료 출처를 제공합니다.
